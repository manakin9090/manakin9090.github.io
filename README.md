# Welcome to ImX!

안녕하세요! 세상에 단 하나뿐인 AI 기반 온디맨드 3D 굿즈와 14K 주얼리를 선보이는 **ImX**의 공식 블로그에 오신 것을 환영합니다.

이곳은 ImX의 브랜드 스토리, 마스코트 '임미(Immi)'의 탄생 과정, 그리고 다채로운 제작 비하인드를 기록하고 아카이빙하는 공간입니다.
